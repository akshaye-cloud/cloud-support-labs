# Networking Basics

## What is a Server?

A server is a system that provides services or resources to other computers over a network.

Example:
Web servers host websites.

---

## IP Address

An IP address uniquely identifies devices in a network.

Example:

192.168.1.1

---

## Public vs Private IP

Public IP

Accessible from the internet.

Example:

54.210.120.11

Private IP

Used inside private networks.

Example:

10.0.0.10

---

## DNS

Domain Name System converts domain names into IP addresses.

Example:

google.com → 142.250.183.46

---

## HTTP

Protocol used to transfer web pages.

Port:

80

---

## HTTPS

Secure version of HTTP using encryption.

Port:

443

---

## Ports

Ports allow multiple services on a server.

Common ports:

22 → SSH  
80 → HTTP  
443 → HTTPS

---

## Firewall

A firewall controls network traffic and protects servers from unauthorized access.

Example:

AWS Security Groups
